## Use notes
- The guide is primarily built for B2B SaaS with some sales-led motion.
- ARR is usually the preferred top-line recurring metric, but MRR or CARR can be substituted when more appropriate.
- Not every metric is derivable from every upload. Use `input-requirements.md` before calculating.
